A Streamlit web app that predicts whether a tech industry employee will seek mental 
health treatment using Machine Learning. Trained 6 models — Logistic Regression, 
Decision Tree, Random Forest, Gradient Boosting, SVM, and KNN on the Mental Health 
in Tech Survey dataset. Features full EDA, confusion matrix, ROC curve, feature 
importance, and a live prediction form. Built with Python, Scikit-learn, and Plotly.
